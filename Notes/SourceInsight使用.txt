
1. 常用窗口作用 及其使用
    1. Clip Window
        保存了最近的复制 剪切的内容
        还有一个功能不太清楚
    2. Relation Window
        显示了与当前函数或变量相关的东西
        调用了那些函数，变量在那个地方被引用了。
    3. Context Window
        显示了当前变量，类型，类等的具体内容，或定义
        Title则显示为内容所属的文件。

    4. Project Window
        Project窗口显示的是内容比较丰富。
        1.工程中所有的文件
        2.工程的Explorer树目录
        3.当前的文件按照文件类型分类列出
        4.工程中所有的符号的信息
        5.符号分类显示

2. 常用快捷键的使用
    Alt + ,     后退
    Alt + .     前进
    Ctrl + ,    后退
    Ctrl + .    前进    
    Ctrl + 鼠标点击符号 可以跳转到符号的定义地方
    
    Ctrl + =    跳转到定义处
    Alt + /     弹出窗口，显示符号的详细信息，比如函数的实现
    F7          弹窗显示工程所有的符号，可以输入匹配
    F8          弹窗显示当前文件的所有符号
    Ctrl + /    搜索符号 在工程中 当前文件等
    F5          跳转到指定行
    
    Ctrl + M    设置书签
    
    Ctrl + F    搜索字符串
    F3          向前搜索
    F4          向后搜索
    
    Shift + F8  高亮显示选中符号
3. 小技巧
    Project  ->  Synchronize Files 用于同步工程文件，搜索其中的符号等信息
        这样可以方便实现跳转等功能。但是同步文件代价比较大
